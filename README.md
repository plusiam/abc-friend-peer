# 🌈 ABC 친구 도우미

초등학생을 위한 또래 상담 도구 - 친구의 마음을 이해하고 도와주는 4단계 상담 시스템

## 📋 프로젝트 개요

ABC 친구 도우미는 초등학생들이 친구의 고민을 체계적으로 상담할 수 있도록 도와주는 웹 기반 교육 도구입니다. ABC 모델(A: 사실, B: 생각, C: 결과)을 기반으로 한 4단계 상담 과정을 통해 아이들이 또래의 마음을 이해하고 효과적으로 도움을 줄 수 있습니다.

## ✨ 주요 기능

### 🎯 4단계 상담 프로세스
1. **💚 마음 공감하기** - 친구의 감정을 파악하고 이해하기
2. **💬 공감 표현하기** - ABC 모델을 활용한 따뜻한 공감 표현 생성
3. **✨ 도움 찾기** - 새로운 생각과 구체적인 해결방법 제시
4. **🎁 격려하기** - 따뜻한 응원 메시지와 실천 약속

### 🎨 핵심 특징
- **상황별 맞춤 체크리스트**: 학교생활, 친구관계, 가족관계, 감정과 자신감 영역별 맞춤형 도움 제공
- **직관적인 UI/UX**: 초등학생도 쉽게 사용할 수 있는 친화적인 인터페이스
- **자동 저장 기능**: 작업 중 데이터 손실 방지를 위한 자동 저장
- **다양한 출력 옵션**: 상담 결과를 이미지, PDF, 인쇄로 저장 가능
- **반응형 디자인**: 태블릿, 스마트폰 등 다양한 기기에서 최적화된 사용 경험

## 🎓 교육적 가치

### 학습 목표
- **공감 능력 향상**: 친구의 감정을 이해하고 공감하는 능력 개발
- **문제 해결 능력**: 체계적인 접근을 통한 문제 해결 사고력 증진
- **의사소통 기술**: 효과적인 상담과 소통 기술 습득
- **정서 지능 개발**: 자신과 타인의 감정을 인식하고 조절하는 능력 향상

### 활용 방안
- **학급 활동**: 또래 상담 프로그램의 일환으로 활용
- **상담 교육**: 학생 상담자 양성 프로그램에서 실습 도구로 사용
- **가정 교육**: 부모-자녀 간 소통 증진 도구로 활용
- **교사 연수**: 상담 기법 교육을 위한 시뮬레이션 도구로 활용

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS
- **Font**: Google Fonts (Noto Sans KR, Jua)
- **Libraries**: 
  - html2canvas (이미지 생성)
  - jsPDF (PDF 생성)
- **Storage**: LocalStorage (자동 저장)

## 📱 사용 방법

### 1. 기본 설정
1. 상담자(도와주는 친구)와 상담받는 친구의 이름을 입력합니다.
2. 고민 사례를 선택하거나 직접 입력합니다.

### 2. 4단계 상담 진행
1. **1단계**: 친구의 감정을 선택하고 공감합니다.
2. **2단계**: ABC 모델을 활용해 공감 표현을 만듭니다.
3. **3단계**: 상황에 맞는 새로운 생각과 구체적인 도움 방법을 제시합니다.
4. **4단계**: 격려 메시지를 작성하고 실천 약속을 합니다.

### 3. 결과 저장
- 상담 완료 후 결과를 이미지, PDF, 또는 인쇄로 저장할 수 있습니다.

## 💡 주요 기능 상세

### 상황별 맞춤 체크리스트
각 고민 상황에 따라 다른 체크리스트가 제공됩니다:

#### 🏫 학교생활
- **도움되는 생각**: "한 번의 시험으로 모든 것이 결정되지 않아", "실수는 배움의 기회가 될 수 있어" 등
- **구체적인 방법**: "어려운 문제는 선생님께 질문하기", "매일 조금씩 공부 계획 세우기" 등

#### 👫 친구관계
- **도움되는 생각**: "친구도 실수할 수 있어", "대화로 해결할 수 있어" 등
- **구체적인 방법**: "친구와 솔직하게 대화하기", "먼저 사과하거나 이해하려 노력하기" 등

#### 👨‍👩‍👧‍👦 가족관계
- **도움되는 생각**: "가족도 완벽하지 않아", "시간을 두고 대화하면 이해할 수 있어" 등
- **구체적인 방법**: "차분할 때 마음을 전달하기", "가족과 함께 시간 보내기" 등

#### 💭 감정과 자신감
- **도움되는 생각**: "이런 감정을 느끼는 건 자연스러운 거야", "시간이 지나면 나아질 거야" 등
- **구체적인 방법**: "믿을 만한 어른에게 이야기하기", "좋아하는 활동하며 기분 전환하기" 등
